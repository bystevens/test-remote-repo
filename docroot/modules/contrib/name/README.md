# Name Field

## Introduction

Name is a module that provides a multi-component name element/field.

Refer to the [project page](https://www.drupal.org/project/name) for an up-to-date list of features.

## Requirements

* Field module

You will require the Field UI module to add fields to content.

## Installation

* Standard installation with no dependencies
* See [Drupal installation guide](http://drupal.org/node/70151) for further information

## Upgrading

* **Drupal 6**: No upgrade path available
* **Drupal 7**: Data migration is supported

## Related Modules

No directly comparable modules exist in Drupal 8.

* **Paragraphs module** can be used for composite fields
* **Real Name module** can also control the display of alternative fields other than the user's login name

## Configuration

There are lots of configurable options. If anything is not clear, submit a support request.
